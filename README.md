# 📄 PDF RAG Chatbot using Google Gemini and LangChain

An elegent, highly responsive full-featured Retrieval-Augmented Generation (RAG) Chatbot application built using **Python**, **Streamlit**, **Google Gemini**, and **LangChain**. 

This application enables developers, researchers, and general users to upload any PDF document, index its text contents locally, and query the documentation naturally. The backend leverages semantic vectors and restricted context pipelines to ensure that every answer generated by the Google Gemini LLM is grounded to facts explicitly present within the document—completely avoiding generative hallucination.

---

## 🚀 Key Features

* **Instant Document Indexing**: Process complete PDF documents, splitting them on page-levels and section-levels without memory leaks.
* **Semantic Vector Pipeline**: Automatically chunks documents into characters (`chunk_size=800`, `chunk_overlap=100`) and encodes them with Google's state-of-the-art embedding engines.
* **Deterministic RAG Retrieval**: Fetches the top 3 most semantically aligned context chunks using local **FAISS** vector databases before prompting.
* **Anti-Hallucination Safe-Guards**: Formulates strict prompt guidelines restricting answers to retrieved scopes. If the question cannot be answered from the document, it safely replies:  
  *"The uploaded PDF does not contain enough information to answer this question."*
* **Interactive Source Telemetry**: Expand and view the exact raw context chunks and their source page references retrieved from the document.
* **Persistent Session Memory**: Cached states prevent FAISS rebuilds on page re-renders, enabling rapid turn-around for multiple queries on the same document.

---

## 🛠️ Technology Stack

* **Streamlit**: Web-based frontend framework for quick and beautiful interactive user interfaces.
* **LangChain**: Modern chaining orchestrator to build document indexing pipelines and vector chains safely.
* **Google Gemini (`gemini-2.5-flash`)**: High-speed, high-quality multimodal large language model.
* **GoogleGenerativeAIEmbeddings**: Dense semantic text representer to encode information cleanly.
* **FAISS (CPU)**: Meta's open-source library for efficient similarity search and clustering of dense vectors.
* **PyPDF**: Highly robust local PDF text parser and extractor.

## 🖥️ Local Installation and Setup

Follow these simple steps to run this application locally on your machine:

### 1. Prerequisite
Ensure that you have **Python 3.11** or a higher stable version installed. Verify via:
```bash
python3 --version
```

### 2. Clone the Repository
```bash
git clone https://github.com/your-username/pdf-rag-chatbot.git
cd pdf-rag-chatbot
```

### 3. Create a Virtual Environment
Initialize a clean Python virtual environment:
```bash
# On macOS/Linux
python3 -m venv .venv
source .venv/bin/activate

# On Windows
python -m venv .venv
.venv\Scripts\activate
```

### 4. Install Dependencies
Install the required packages defined in `requirements.txt`:
```bash
pip install -r requirements.txt
```

### 5. Set up Environment Variables
Duplicate `.env.example` as `.env` and fill in your custom API Key:
```bash
cp .env.example .env
```
Inside your new `.env` file, replace placeholder values with your real Gemini API key:
```env
GOOGLE_API_KEY=AIzaSy...yourKeyHere...
```

### 6. Run the Application
Start the local Streamlit development server:
```bash
streamlit run app.py
```
This launches the app in your primary web browser automatically, usually bound to `http://localhost:8501`.

## ⚙️ Deploy to Production

This workspace is structured to be deployed easily to any standard platform:

### Deploy to Streamlit Community Cloud
1. Push your code directly to a public GitHub repository.
2. Sign in to [Streamlit Share](https://share.streamlit.io/).
3. Click "New App", select your repository, branch (`main`), and set the Main file path to `app.py`.
4. In Advanced Settings, add your Gemini API key in the Secrets input box:
   ```toml
   GOOGLE_API_KEY = "your-actual-api-key"
   ```
5. Click **Deploy**!

### Deploy to Render / Railway / Cloud Run
Ensure that you bind your container port using the dynamic environment variables and pass `GOOGLE_API_KEY` into your secret deployment environment dashboard securely.
